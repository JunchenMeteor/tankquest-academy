# AI Service

AI service placeholder. The planned stack is FastAPI + LangChain/LangGraph.

AI may:

- Generate question drafts
- Explain wrong answers
- Suggest difficulty adjustments
- Summarize parent reports

AI must not:

- Decide correctness
- Grant rewards
- Upgrade tanks
- Write directly to business tables
- Provide medical or psychological diagnosis
