# TankQuest Academy v0.3.0

发布日期：2026-07-16

## 主要变化

- 新增可选 FastAPI AI 学习辅助服务、严格结构化契约、LangChain 供应商适配和确定性模板。
- 新增儿童安全错题解释；后端继续独立判题、记录答案并发放奖励。
- 新增后端裁决的自适应练习推荐，受家长上限、单次一级和真实已发布内容约束。
- 新增家长学习教练摘要，只使用聚合学科与技能指标，进步趋势和文案由主后端控制。
- 新增诊断、能力标签、焦虑措辞、外链、隐私请求和多句输出的 FastAPI/Nest 双层拒绝。
- 部署扩展为 Web、API、AI、PostgreSQL 四容器，并通过 CI 生成的离线 Python wheelhouse 避免服务器依赖下载失败。

## 验证

- API 73、Web 42、Shared 6、AI 53 项单元和契约测试通过。
- PostgreSQL migration、重复 seed 和 5 项 Playwright 学习/家长流程通过。
- 格式、Prisma schema、lint、全仓 typecheck 和生产构建通过。
- API、Web、AI 三套运行时容器检查通过。
- preview/release 页面、健康接口、家长摘要和四容器在受保护发布流程中验证。

## 明确不包含

- 把 AI 作为判题、奖励、升级、最终难度或正式内容发布的权威来源。
- 自动发布 AI 题目草稿或绕过人工内容审核。
- 正式多家庭鉴权、多孩子管理和完整家长后台。
- 用自动化替代真实儿童和家长试用，或替代真实外部模型的长期评估。
- 商城、抽卡、付费数值、多人联机、2.5D/3D 和客户端打包。
