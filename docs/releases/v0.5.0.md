# TankQuest Academy v0.5.0

发布日期：2026-07-17

## 主要变化

- 新增复用现有 React、Phaser 4.2.1 和客户端契约的 Tauri 2 Windows 技术预览，不复制游戏、学习或结算规则。
- 新增显式 PlatformClient 边界、Browser/Tauri adapter 和 fail-closed 平台选择；Windows WebView 不注册 PWA Service Worker。
- development、preview、release 使用独立 API origin、Vite mode 和最小 `connect-src` CSP，跨环境配置会阻止启动。
- Windows 启动前验证配置、native bridge、网络和 API 健康；失败时显示本地化安全状态，不创建正式会话、奖励或升级。
- GitHub Actions 新增固定 Rust 1.97.0 的 Windows build、fmt、release Clippy、未签名 NSIS 打包和响应窗口烟测。
- Linux Core 与 Windows desktop 并行运行，受保护 Verify 必须同时收到两边成功；npm 和 Rust 缓存按锁文件复用。
- 官方 Actions 升级到 Node 24-backed 当前主版本，AI wheelhouse 使用内容寻址的服务器缓存，命中时跳过约 22 MB artifact 下载。

## 验证

- API 80、Web 105、Shared 9、AI 53 项单元与契约测试通过。
- PostgreSQL migration、重复 seed 和 7 项 Playwright 学习、家长、触控与离线边界流程通过。
- Windows 干净 runner 完成配置验证、Web preview 资源、Rust fmt/release Clippy、NSIS 构建和响应窗口烟测。
- 技术预览安装器 artifact 为 2,274,328 bytes，单独的 Rust 编译缓存不进入发布或部署 artifact。
- 格式、资源/PWA 策略、Prisma schema、lint、全仓 typecheck、生产构建、性能预算和 API/Web/AI 三套容器检查通过。
- preview/release 的 Web、API/AI 健康、家长摘要、PWA、资源 manifest/hash 和四容器由受保护发布流程验证；任一门禁失败都不得创建 GitHub Release。

## 明确不包含

- Windows 代码签名、面向公众的正式安装器分发、自动更新、增量补丁或 Windows 商店发布。
- 离线正式对局、本地权威存档、离线奖励/升级或学习记录后台同步。
- 完整 3D、自由镜头、3D 物理、多人联机、Android/iOS 打包或应用商店提交。
- 正式多家庭鉴权、真实儿童家庭可用性结论，或用 hosted runner 取代真实 Windows 设备验证。
