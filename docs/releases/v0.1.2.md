# TankQuest Academy v0.1.2

发布日期：2026-07-15

## 主要变化

- 新增由火力等级推导的炮弹穿深。
- 新增由装甲等级推导的正面、侧面和背面装甲。
- 命中结果根据区域、入射角、穿深和有效装甲确定为跳弹、未击穿或击穿。
- 玩家和敌方坦克使用相同的确定性命中规则。
- 增加儿童可理解的命中标签和 HUD 穿深/装甲读数。
- 战斗日志增加区域、角度、穿深、有效装甲、伤害和剩余生命值。
- 保留敌方炮弹连续命中后游戏循环继续运行的回归测试。

## 验证

- API 单元测试通过。
- Web 战斗和运行时单元测试通过。
- Shared 契约测试通过。
- Playwright 完整学习闭环和连续受击回归通过。
- 格式、Prisma schema、lint、typecheck 和生产构建通过。
- preview 和 production Docker 健康检查通过后发布。

## 明确不包含

- 随机暴击和部件损坏。
- 炮弹类型和弹药库存。
- 真实军用穿甲公式或成人化暴力反馈。
- Phase 2 多坦克、涂装和学习记录功能。
