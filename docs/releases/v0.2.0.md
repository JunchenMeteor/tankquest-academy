# TankQuest Academy v0.2.0

发布日期：2026-07-15

## 主要变化

- 新增轻型、中型、重型三辆原创训练坦克，并让属性差异进入实际对局。
- 新增每辆坦克两套原创涂装，装备选择由后端持久化。
- 新增按孩子、学科和技能聚合的学习记录与查询 API。
- 新增与儿童游戏分离的 30 天家长报告基础版。
- 新增英语词义匹配和方向判断任务，复用后端判题、结算和记录链路。
- 默认界面语言为英语，可切换简体中文；训练主题选择刷新后保留。
- 部署工作流统一为 `TankQuest Preview and Release`，部署 Job 统一为 `TankQuest / Deploy preview or release`。

## 验证

- API、Web 和 Shared 单元测试通过。
- PostgreSQL migration、重复 seed 和三学科 Playwright 闭环通过。
- 格式、Prisma schema、lint、seed typecheck、应用 typecheck 和生产构建通过。
- API/Web Docker 运行时镜像构建通过。
- preview 和 release 页面、健康接口与 Compose 容器在发布流程中验证。

## 明确不包含

- 正式多家庭鉴权、多孩子管理和完整家长后台。
- AI 学习教练、AI 出题和 AI 报告总结。
- 商城、抽卡、付费数值、多人联机、3D 和客户端打包。
- 用自动化替代真实儿童和家长家庭试用。
