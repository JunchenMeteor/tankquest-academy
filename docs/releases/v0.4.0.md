# TankQuest Academy v0.4.0

发布日期：2026-07-16

## 主要变化

- 新增后端管理的版本化资源清单、公开只读 manifest API，以及校验同源 URL、依赖、大小、SHA-256 和实际字节的 Web `AssetClient`。
- 三辆玩家坦克与 scout、medium、heavy 敌人升级为高识别度分层轮廓，涂装在选择页和下一局中保持一致。
- 三类训练地图增加主题化等距地面、立体障碍、边界和深度层，同时保留既有 Arcade Physics 坐标、碰撞和伤害规则。
- 键盘、鼠标、指针和触控统一映射为游戏命令；粗指针设备提供移动、转向、瞄准和开火控制。
- 新增可安装 PWA 应用壳、版本化图标和明确离线提示；Service Worker 永不缓存 API、报告、会话、答案、奖励或进度。
- 新增可执行 Service Worker 缓存策略校验、离线业务操作守卫、gzip/视觉资源预算和部署后公开契约探针。

## 验证

- API 80、Web 72、Shared 9、AI 53 项单元与契约测试通过。
- PostgreSQL migration、重复 seed 和 7 项 Playwright 学习、家长、触控与离线边界流程通过。
- 格式、资源/PWA 策略、Prisma schema、lint、全仓 typecheck、生产构建和性能预算通过。
- API、Web、AI 三套运行时容器检查通过。
- preview/release 的 Web、API/AI 健康、家长摘要、PWA 入口、资源 manifest/hash 和四容器由受保护发布流程验证；任一门禁失败都不得创建 GitHub Release。

## 明确不包含

- 完整 3D、自由镜头、3D 物理、Three.js、Unity、Godot 或原生应用商店发布。
- 离线正式对局、离线奖励结算、学习记录后台同步或缓存任何儿童业务数据。
- 对象存储/CDN、大型 GLB、高清视频、商业模型复刻或自动发布儿童可见生成式美术。
- 商城、抽卡、付费数值、多人联机、正式多家庭鉴权或用自动化替代家庭试用。
