# 游戏数值与经济系统

## 1. 核心原则

- 学习表现影响奖励。
- 坦克升级改善手感，但不能替代学习。
- 儿童模式不做强付费刺激。
- 数值简单可解释。

## 2. 坦克属性

属性范围 1-5：

```text
火力
机动
装甲
隐蔽
视野
```

每升一级必须有可感知变化。

## 3. 奖励类型

```text
星星：通关评价和解锁
零件：升级属性
徽章：学习成就
训练点：通用软货币
```

## 4. 星级规则示例

```text
1 星：完成主要目标
2 星：答题正确率 >= 60%
3 星：答题正确率 >= 80% 且坦克未被击毁
```

## 5. 升级规则示例

```text
Lv1 -> Lv2: 2 个对应零件
Lv2 -> Lv3: 4 个对应零件
Lv3 -> Lv4: 8 个对应零件
Lv4 -> Lv5: 12 个对应零件
```

## 6. 难度调整

连续失败：

- 减少敌人。
- 增加提示。
- 降低题目干扰项。

连续成功：

- 增加题目难度。
- 增加敌人组合。
- 提高奖励上限。

## 7. 防数值失控

- 每日奖励可设软上限。
- 坦克属性有等级上限。
- 关卡有推荐战力但不能强卡。
- 成人模式和儿童模式奖励可分池。
