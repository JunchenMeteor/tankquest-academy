# AI 服务与安全设计

## 1. AI 服务定位

AI 服务是学习辅助系统，不是游戏裁判。

AI 可以做：

- 题目草稿生成
- 错题解释
- 个性化学习建议
- 家长报告总结
- 趣味心理测试文案
- 难度调整建议

AI 不可以做：

- 判定答案正确性
- 发放奖励
- 修改坦克属性
- 决定是否通关
- 直接写业务数据库
- 自由陪聊
- 心理诊断或医学建议

## 2. 服务架构

```text
主后端
  -> AI Gateway
      -> FastAPI AI Service
          -> LangChain / LangGraph
          -> LLM Provider
```

## 3. AI 模块

```text
QuestionGeneratorChain
ExplanationChain
DifficultyPlannerChain
ParentReportChain
FunPersonalityTestChain
SafetyGuard
```

## 4. 后端校验

主后端必须校验：

- JSON schema 正确
- 答案唯一
- 正确答案可验证
- 难度在允许范围内
- 内容适合用户年龄
- 不包含成人内容
- 不包含医疗诊断
- 不包含外链诱导
- 不包含暴力煽动

## 5. 儿童安全边界

儿童模式 AI 只能输出：

```text
题目解释
学习鼓励
下一步练习建议
家长报告摘要
```

儿童模式 AI 禁止：

```text
自由聊天
情绪诊断
心理测试
成人话题
真实战争宣传
外部链接
索要隐私
```

## 6. 心理测试边界

心理测试仅允许在成人休闲模式或家长模式出现。必须声明：

```text
该测试仅用于娱乐和自我观察，不构成医学、心理诊断或治疗建议。
```

## 7. 降级策略

AI 服务失败时，游戏不能中断。

- 使用预置题库
- 使用模板解释
- 跳过 AI 报告生成
- 后台稍后异步生成报告
- 返回友好提示

## 8. 错题解释契约

`POST /v1/internal/wrong-answer-explanations` 只接收年龄段、语言、学科、技能、难度、题干、孩子本次选择和主后端已经确定的正确答案。请求不得包含 child、session、question、answer 或学习记录 ID。

主后端必须先完成判题和学习记录写入，再按家长开关调用 AI。AI 响应回显的 `correctAnswer` 必须与主后端输入完全一致；否则使用题库正式解释。AI 不能修改 `correct`、弹药、奖励或进度。

## 9. 自适应练习契约

`POST /v1/internal/practice-recommendations` 只接收年龄段、学科、技能和聚合的尝试数、正确率、平均耗时、完成局数，以及主后端计算的允许难度范围。请求不得包含业务 ID、身份字段、原始答案、题目或对局事件。

主后端负责选择薄弱技能、计算允许范围、限制单次最多变化一级，并把建议映射到家长允许且真实已发布的关卡。AI 不得返回 `levelId`；学科或技能回显不一致时拒绝，难度越界时由后端钳制。AI 不可用时使用同一套确定性策略，结算和奖励不受影响。

## 10. 家长报告摘要契约

`POST /v1/internal/parent-report-summaries` 只接收语言、完成局数、答题总数，以及最多五个学科和五个技能的聚合次数、正确率、平均耗时、当前难度与主后端趋势标签。请求不得包含 childId、姓名、昵称、出生年份、日期、数据库 ID、原始答案、题目、session、游戏事件或战斗日志。

主后端先用前后半窗口计算 `improving`、`steady`、`needs-practice` 或 `insufficient-data`，AI 只生成练习内容、进步证据、需要支持和下一步建议四段短文。没有 `improving` 信号时不得声称明显进步。诊断、能力或人格标签、医学判断、焦虑式措辞、外链和隐私请求会触发本地确定性降级；儿童游戏界面不得显示家长摘要。
