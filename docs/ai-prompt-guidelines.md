# AI Prompt 与审核规范

## 1. Prompt 原则

- 输出必须结构化。
- 不允许自由发挥业务规则。
- 儿童内容必须安全、正向、简短。
- 心理测试只允许娱乐模式。

## 2. 出题 Prompt 输入

```json
{
  "ageGroup": "6-8",
  "subject": "math",
  "skill": "addition_carry",
  "difficulty": 2,
  "locale": "zh-CN"
}
```

## 3. 出题输出 Schema

```json
{
  "question": "8 + 7 = ?",
  "answers": ["12", "15", "17"],
  "correctAnswer": "15",
  "explanation": "8 加 7 等于 15。"
}
```

## 4. 错题解释要求

解释必须：

- 不责备。
- 不超过儿童理解长度。
- 给出一步提示。
- 可以用图形或拆分思路。
- 最多两句、320 个字符。
- 必须原样回显主后端提供的正确答案。
- 不得索要隐私、提供外链或改变判题和奖励。

## 5. 家长报告要求

报告包含：

```text
练习内容
正确率
明显进步
需要加强
建议下一步
```

不包含：

```text
心理诊断
负面标签
医学判断
过度焦虑表达
```

## 6. 审核规则

AI 输出入库前必须检查：

- 正确答案是否唯一。
- 数学答案是否正确。
- 文案是否适龄。
- 是否含敏感内容。
- 是否符合 schema。
