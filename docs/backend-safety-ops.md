# 家长控制、运营后台与数据指标

## 1. 家长控制

```text
每日可玩时长
单次游玩时长
休息提醒间隔
允许题库类型
允许难度范围
是否允许 AI 错题解释
是否允许趣味心理测试
是否允许联网同步
```

儿童账号默认不能进入成人模式、不能看到心理测试、不能自由和 AI 聊天、不能打开外链、不能看到真实军事暴力内容。

## 2. 内容版权策略

坦克采用“经典风格启发 + 原创命名 + 原创涂装”。

允许：

- 保留大致轮廓特征
- 保留炮塔/炮管/履带比例特征
- 儿童化改造
- 原创名称
- 原创涂装

避免：

- 真实军徽
- 真实编号
- 纳粹/军队符号
- 真实战役皮肤
- 高度还原商用模型
- 直接使用受版权保护素材

## 3. 防沉迷和节奏

```text
单局 3-5 分钟
连续 20-30 分钟提示休息
每日上限家长可配置
连续失败自动降难度
答错不羞辱，只给提示
通关后鼓励休息或总结
```

## 4. 后台管理系统

后台模块：

```text
题库管理
关卡管理
坦克管理
涂装管理
奖励规则管理
用户/孩子档案查看
学习报告查看
AI 生成内容审核
资源管理
多语言文案管理
主题管理
```

后台角色：

```text
admin: 全权限
content_editor: 题库/文案/关卡
designer: 坦克/涂装/资源
parent: 只看自己孩子
```

配置发布状态：

```text
draft
published
archived
```

核心指标：

```text
通关率
失败原因

题目耗时
关卡卡点
坦克使用率
升级转化
连续游玩天数
家长报告查看率
```

## 6. 商业模式边界

儿童产品不建议抽卡、限时逼氪、随机宝箱、付费变强和儿童诱导消费。

推荐买断制、家庭订阅、扩展题库包、坦克主题包、亲子挑战包、学校/机构版。
