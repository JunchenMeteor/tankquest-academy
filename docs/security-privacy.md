# 安全与隐私规范

## 1. 基本原则

- 儿童数据最小化。
- 家长可控。
- AI 不接触不必要隐私。
- 日志脱敏。
- 默认关闭成人内容。

## 2. 儿童数据

允许保存：

```text
昵称
年龄段
学习记录
关卡进度
坦克收藏
家长设置
```

避免保存：

```text
真实姓名
学校
详细地址
身份证件
精确定位
无关聊天内容
```

## 3. 家长控制

家长必须能：

- 查看孩子进度。
- 设置游玩时长。
- 控制题库范围。
- 禁用 AI 解释。
- 禁用成人模式。
- 请求删除孩子数据。

## 4. AI 安全

AI 请求只传必要上下文：

```text
年龄段
题目
错误答案
学习技能标签
```

禁止传：

```text
真实姓名
邮箱
家庭信息
详细行为日志
```

## 5. 日志脱敏

日志不记录：

- 密码。
- token。
- 真实联系方式。
- 完整儿童隐私数据。
- AI 原始敏感输入。

## 6. 安全措施

- JWT 短期 token + refresh token。
- 密码哈希。
- 后台 RBAC。
- 管理端审计日志。
- API 限流。
- 上传资源类型校验。
- 禁止前端直接访问数据库。
